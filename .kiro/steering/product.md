# Product Overview

Scensei（Scent + Sensei）は、香水ソムリエAIキャラクターを実装したWebアプリケーション。VRM形式の3Dアバターと対話し、ユーザーの好みやシーンに合わせた香水の提案を受けることができる。

## Core Capabilities

1. **対話型香水コンサルティング**: Claude AIを搭載したキャラクターがユーザーとの会話を通じて香水を提案
2. **3Dアバター表示**: VRM形式のアバターがリアルタイムで表情・ジェスチャー・リップシンクを実行（男女モデル切替対応）
3. **香水データベース検索**: 開発者の香水レビューデータをMCP経由で検索・参照
4. **会話メモリ**: 短期記憶（セッション内）と長期記憶（ユーザープロファイル）で継続的な対話を実現

## Target Use Cases

- 香水に興味があるが何を選べばよいかわからないユーザーへの提案
- シーン（デート、ビジネス等）や季節に合った香水の相談
- 香水の特徴やノート（トップ/ミドル/ベース）についての質問
- 過去に試した香水をベースにした新しい香水の発見

## Value Proposition

- **キャラクター性**: AITuber-kitベースの3Dアバターにより、チャットボット以上の親しみやすい対話体験
- **パーソナライズ**: 長期記憶により、ユーザーの好みを学習し継続的に精度の高い提案が可能
- **専門知識**: 開発者自身の香水レビューデータベースに基づく具体的な提案
